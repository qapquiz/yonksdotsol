## Problem

`src/app/index.tsx` renders ALL position cards via `ScrollView` + `.map()`. Every `PositionCard` mounts immediately regardless of visibility. With many DLMM positions this causes long mount times and high memory usage.

## Plan

1. Install `@shopify/flash-list` (or `@legendapp/list`)
2. Refactor `src/app/positions/index.tsx` — flatten the nested `positionsEntries.map()` into a flat data array for the virtualizer
3. Move `PortfolioSummary` and out-of-range banner into `ListHeaderComponent`
4. Move bottom padding into `ListFooterComponent`
5. Set `estimatedItemSize` based on `PositionCard` height (~350px)
6. Keep `RefreshControl` on the virtualized list (or use `onRefresh`/`refreshing` props if FlashList supports them)

## Files affected

- `src/app/index.tsx` — swap `ScrollView` for the virtualized list
- `src/app/positions/index.tsx` — return flat data array + header/footer instead of wrapped `View`
