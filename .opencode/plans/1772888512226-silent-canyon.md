## Implementation Notes

### Helius API Usage

- Endpoint: `https://api-mainnet.helius-rpc.com/v0/addresses/{address}/transactions`
- Parameters: `api-key`, `limit: 100`, `before-signature` (pagination)
- Filters: `ADD_LIQUIDITY`, `ADD_LIQUIDITY_ONE_SIDE`, `ADD_LIQUIDITY_BY_STRATEGY`, `ADD_LIQUIDITY_ONE_SIDE_PRECISE`, `TRANSFER`

### Native SOL Detection

- Filters transfers from owner address
- Only counts amounts between 10M-100M lamports (avoids rent and dust)

### Caching Strategy

- Cache key: `helius_initial_deposits:{positionAddress}`
- TTL: 15 minutes (existing `CACHE_TTL.INITIAL_DEPOSITS`)
- CacheManager instance: singleton pattern

### Error Handling

- Returns empty arrays and 0 on error
- Logs errors to console
- UI shows loading/error states appropriately
