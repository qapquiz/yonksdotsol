# Plan: Use Comet's getUpnl for SOL-denominated UPNL

Replace the current USD-based UPNL calculation with comet's `getUpnl` function, which returns values denominated in SOL.

## Current State

- `useInitialDepositsHelius` hook fetches initial deposits via Helius API
- UPNL calculated in USD: `current - initial`
- Display: `$X.XX (+X.XX%)`
- Scope: Per-position

## Target State

- Use comet's `getUpnl(walletAddress, connection, heliusApiKey)`
- UPNL denominated in SOL
- Display: `X.XX SOL (+X.XX%)`
- Scope: Wallet-level (aggregates all positions)

export function getUpnl(params: GetUpnlParams): Promise<UpnlResult | null>

  const sign = upnl >= 0 ? '+' : ''
  return `${sign}${Math.abs(upnl).toFixed(4)} SOL (${sign}${percent.toFixed(2)}%)`
}

## Important Notes

- Comet only calculates SOL values for **SOL-paired positions** (where one token is native SOL)
- UPNL is wallet-level aggregation, not per-position
- Non-SOL pairs return `0` for SOL-denominated values
