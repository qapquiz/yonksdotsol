# Plan: Upgrade metcomet and Use getUpnlPerPosition

## Overview

Replace custom `useInitialDepositsHelius` + manual uPNL calculation with `metcomet@0.2.2`'s `getUpnlPerPosition` function, which returns per-position uPNL data in a single API call.

## Why This Change

- **Efficiency**: `getUpnlPerPosition` fetches uPNL for all positions in ONE call (vs. N calls for N positions)
- **Accuracy**: metcomet handles historical price lookups for initial deposit valuation
- **Less code**: Remove ~170 lines of custom Helius transaction parsing
- **Richer data**: Also returns `unclaimedFeesInSol`, `upnlWithFees`, and `upnlWithFeesPercent`

## Implementation Steps

### 1. Upgrade metcomet

```bash
bun add metcomet@^0.2.2
```

### 2. Create new hook: `useUpnlPerPosition.ts`

```typescript
interface PositionUpnlData {
  pairAddress: string
  positionAddress: string
  initialDepositInSol: number
  currentValueInSol: number
  unclaimedFeesInSol: number
  upnl: number
  upnlWithFees: number
  upnlPercent: number
  upnlWithFeesPercent: number
}

interface UseUpnlPerPositionResult {
  data: Map<string, PositionUpnlData> | null // keyed by positionAddress
  isLoading: boolean
  error: Error | null
}

export function useUpnlPerPosition(walletAddress: string, enabled?: boolean): UseUpnlPerPositionResult
```

### 3. Cache Strategy

- Key: `upnl_per_position:${walletAddress}`
- Wallet-level fetch, but data accessible per-position
- TTL: 15 minutes (same as current `INITIAL_DEPOSITS`)

### 4. Update PositionsList (`src/app/positions/index.tsx`)

- Call `useUpnlPerPosition(walletAddress)` once at list level
- Pass `upnlData: Map<string, PositionUpnlData>` to each `PositionCard`

### 5. Update PositionCard props

```typescript
interface PositionCardProps {
  position: PositionInfo
  lbPositionIndex?: number
  upnlData?: PositionUpnlData | null // NEW: passed from parent
}
```

### 6. Simplify PositionCard

- Remove `useInitialDepositsHelius` hook call
- Remove `initialDepositValue`, `currentValueNumber` calculations
- Remove `calculateUPNLValue`, `calculateUPNLPercentage` usage
- Use `upnlData.upnl` and `upnlData.upnlPercent` directly

### 7. Update PositionHeader

- Keep existing props (`upnlValue`, `upnlPercentage`, `upnlIsSol`)
- PositionCard passes `upnlData?.upnl`, `upnlData?.upnlPercent`

### 8. Clean up Unused Code

- Delete `src/hooks/positions/useInitialDepositsHelius.ts`
- Delete `src/utils/positions/helius-initial-deposit.ts`
- Delete `src/hooks/positions/useMetCometUpnl.ts`
- Remove `METCOMET_UPNL_KEY` and `getMetCometUpnlKey` from cacheKeys
- Remove unused `INITIAL_DEPOSITS_KEY` if no longer used

## Data Flow (After Changes)
